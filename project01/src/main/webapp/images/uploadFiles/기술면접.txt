 - 스택 FILO 선입출된 원소가 가장마지막에 서비스받는구조
   큐 FIFO 선입출된 원소가 가장먼저 서비스받는구조

   어레이 연속된 공간을 잡고 이에대한 접근을 INDEX로 할수 있는것

   리스트 배열을 기반으로 구현되었으며 중복된 데이터의 저장을 막지 않고, 순차적 저장
   추가/삭제시 느리지만 인덱스접근시 빠르다 (링크드리스트와 반대)

   맵 원소를 저장하고 그것을 키를 이용하여 빠르게 찾을수 있도록 하는것 - 리스트나 배열처럼 순차적 요소로 값을 구하지않음
   해쉬맵 맵의 기능을 가지고 있고 키값과 밸류값을 설정할수 있다

클래스와 인터페이스

 - 추상클래스 인스턴스를 직접 생성하지 못하고 생성자를 사용할수 없다
   
   인터페이스 클래스나 컴포넌트가 제공한는 메소드들을 선언한것 - 인터페이스는 CLASS에 상속이 안되고 클래스는 인터페이스에 상속이 가능합니다
   
   컨포넌트 어떠한 일부분에서 그 모듈들이 상호작용할수 있게하는 조합
    - JDBC Java Database Connectivity - 자바에서 테이블 형식의 데이터 소스에 액세스 하는데 사용하는 API
      자바서블릿 웹서버 쪽에 작성된 자바프로그램 - 스레드 형태로 실행디므로 다시 읽어들일 필요가 없으며 독립적 플랫폼이다
   스레드 실제 작업을 수행하는 단위
   API 자바의 클래스 라이브러리의 집합체입니다
   라이브러리 언제는 함수를 자유롭게 사용할수 있게끔 모아놓으것

hibernate
